# YouTube概要欄：「寝だめ」は効かない──週末の寝坊では取り戻せない

「週末にたっぷり寝れば、平日の睡眠不足はチャラになる」──そう思っていませんか？
実はこの「寝だめ」、最新の科学研究では効果がないどころか、逆効果になることさえわかっています。
日本人の睡眠時間はOECD加盟33カ国中で最短。多くの人が週末の「寝だめ」に頼っていますが、
コロラド大学の2019年の研究では、寝だめと不足の繰り返しがずっと寝不足よりも代謝に悪い可能性が示されました。
この動画では「睡眠負債」の正体、寝だめが効かない科学的根拠、そして今日からできる正しい解消法を解説します。
