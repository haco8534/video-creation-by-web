# YouTube概要欄：「ストレスは体に悪い」は半分嘘だった

「ストレスは万病の元」──多くの人がそう信じていますよね。
でも、アメリカの約3万人を8年間追跡した大規模調査で、衝撃的な事実が明らかになりました。
ストレスに関連して年間約2万人が早死にしていた──しかし、その原因はストレスそのものではなく、「ストレスは体に悪い」という信念だったのです。
この動画では、ストレスに対する考え方が体の反応をどう変えるのか、チャレンジ反応と脅威反応の違い、オキシトシンの意外な役割、そして本当に怖い慢性ストレスの害まで、科学的根拠に基づいて徹底解説します。
