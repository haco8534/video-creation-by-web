# YouTube概要欄：血液型性格診断に科学的根拠は一切ない

<!-- 以下をそのままYouTubeの概要欄にコピー＆ペーストして使う -->

「A型は几帳面」「B型はマイペース」──日本人の約40%が信じている血液型性格診断。
でも、日米1万人以上の大規模調査で明らかになった衝撃の結論は「最大でも0.3%以下」。
1927年の提唱から100年近い歴史の中で、科学者たちは何度も検証してきましたが、誰一人として血液型と性格の関連を証明できた人はいません。
なぜ根拠がないのにこんなに信じられているのか？その心理メカニズムと、血液型信仰が生む「ブラハラ」の実害まで徹底解説します。
