# ▶ 終わりに

## 【Block 6：まとめ「利己と利他のコインの裏表」】

ずんだもん：さて、今日のまとめなのだ。

ずんだもん：2つ目。行動経済学のゲーム実験と脳科学が、人間の利他性を科学的に実証しているのだ。
