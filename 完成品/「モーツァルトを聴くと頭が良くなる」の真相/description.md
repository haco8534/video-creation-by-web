# YouTube概要欄

「モーツァルトを聴くと頭が良くなる」――この話、一度は聞いたことがあるのではないでしょうか？

しかし、2010年の39研究をまとめたメタ分析で、モーツァルト効果の科学的根拠はほぼ否定されています。元の実験はたった36人の大学生が対象で、IQではなく空間推理テストのみ、しかも効果は15分で消えるものでした。

この動画では、1993年の元祖実験の真実から、メディアの暴走、追試の崩壊、そして音楽の「本当の」知的効果まで、科学的に丁寧に解説します。
