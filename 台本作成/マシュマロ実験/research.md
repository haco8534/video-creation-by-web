## 📌 台本構成への提言

### Block 1（導入）に使うべきファクト
- 実験の基本概要（マシュマロ1個 vs 2個、15分待てるか）→ 視聴者の「知ってる！」感を引き出す
- フック：「この実験が"人生の成功を予測する"って思ってませんか？ 実は、ほぼ間違いだったのだ」

### Block 2（前提知識）に使うべきファクト
- 実験の詳細なプロトコル、3分の1が待てた、SAT 210点差
- 追跡調査の「すごい結果」をしっかり紹介（後で覆すための前振り）

### Block 3（本論①：通説の深掘り）に使うべきファクト
- 認知戦略（歌を歌う、目を覆う、雲に見立てる）
- ホット・クールシステム理論
- fMRI研究（前頭前野 vs 扁桃体）
- ここまでは「すごい実験だ」というトーン

### Block 4（本論②：反論＝最大の意外性）に使うべきファクト ← **動画の山場**
- Watts 2018の大規模再現：効果半減 → 統制で消失
- Kidd 2013：環境の信頼性実験（12分 vs 3分）
- サンプルバイアスの問題（スタンフォード保育園＝富裕層）
- 貧困環境での「合理的判断」の視点

### Block 5（本論③：背景・文化差）に使うべきファクト
- 日米文化差研究（マシュマロ vs プレゼントで結果逆転）
- ミシェル本人の真意（測定ではなくスキル習得）
- if-thenプランニングなどの実用的応用

### Block 6（まとめ）に使うべきファクト
- 「自制心 = 成功」の単純公式は誤り → 環境・信頼・文化が重要
- ミシェル自身が伝えたかったこと：自制心は鍛えられる
- 冒頭フックの回収
