## 📌 台本構成への提言

### リサーチ統合の結論
3方向のリサーチを統合すると、マシュマロ実験の物語は**「通説の形成 → 通説の崩壊 → より深い理解」**という美しい三幕構成を持っている。これは動画の構成にそのまま活用できる。

### ブロックごとの推奨ファクト配置

| ブロック | 推奨ファクト |
|---------|------------|
| **Block 1（導入・フック）** | ミシェルの実験の「有名なイメージ」（子どもがマシュマロを前にして葛藤する姿）を使う。「SAT 210点差」のインパクトで引き込み → 「でも、この話には続きがあるんです」 |
| **Block 2（前提知識）** | 実験の正確な条件（4歳・15分・ビング保育園）を丁寧に解説。ミシェルの経歴（ウィーン→ナチス→渡米）で人間的な厚みを。我慢できた子の面白い戦略（歌・目そらし・リフレーミング）で視聴者を楽しませる |
| **Block 3（通説の解説）** | 追跡調査の結果（SAT 210点差、社会的有能さ、BMI、脳の活動）を段階的に紹介。ホット/クールシステムの説明。「ここまでが、みんなが知っているマシュマロ実験」と区切り |
| **Block 4（反論・意外な事実）** | ★最重要ブロック。ワッツらの再現実験（効果半減→統制で3分の2さらに減少→消滅）。キッドの「信頼の実験」（12分 vs 3分）。「子どもが我慢しないのは意志が弱いからじゃない、環境が信頼できないからだ」 |
| **Block 5（深掘り・なぜそうなるか）** | 「欠乏のマインドセット」理論。日米文化差の面白い発見。再現性の危機と科学の進歩。ミシェル自身の「本当に言いたかったこと」 |
| **Block 6（まとめ）** | 「我慢できる子が成功するのではなく、安心できる環境にいる子が自然と待てる」。「子どもに我慢を教えるより、約束を守る大人になろう」。冒頭の問いの明示的回収 |
