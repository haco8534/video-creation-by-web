# プラセボ効果はどこまで「効く」のか？

偽の手術で膝の痛みが消えた — これ、本当にあった話なんです。
プラセボ効果は「ただの気のせい」だと思っていませんか？

最新の脳科学が明らかにしたのは、偽薬でも脳内でドーパミンや鎮痛物質が実際に放出されるという科学的事実でした。PETスキャンやMRIで確認されたプラセボ効果のメカニズム、薬の値段や色で効果が変わる驚きの実験、そして偽薬だとわかっていても効く「開放型プラセボ」の謎に迫ります。

一方でプラセボにも限界があります。がんの腫瘍には効きません。さらに「ノセボ効果」——医師の否定的な一言が副作用を生み出す怖い現象も解説します。
