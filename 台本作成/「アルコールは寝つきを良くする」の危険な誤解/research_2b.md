# リサーチ 2b：社会的データ・統計（日本の寝酒文化と健康リスク）

## 日本の寝酒文化の実態

### 寝酒習慣の国際比較
- **2004年 10カ国比較調査**: 眠るためにアルコールを使用する割合
  - 日本: **30.3%**（10カ国中最も高い）
  - 10カ国平均: 19.4%
  - 日本人の寝酒率は国際平均の約1.6倍

### 日本人の寝酒頻度
- 週1回以上寝酒をする割合: 男性 **48%**、女性 **18%**
- 男性のほぼ2人に1人が寝酒の習慣あり

### 飲酒習慣（厚生労働省データ）
- 習慣飲酒者（週3回以上、日本酒1合相当以上）の割合:
  - 男性: 1989年 51.5% → 2019年 33.9%（減少傾向）
  - 女性: 1989年 6.3% → 2019年 8.8%（増加傾向）
- 生活習慣病リスクを高める飲酒者: 男性14.1%、女性9.5%

## アルコール依存症の統計

### 有病率
- 2013年全国調査:
  - 生活習慣病リスクを高める飲酒者: **1,036万人**
  - アルコール依存症が疑われる人（AUDIT20点以上）: **112万人**
  - ICD-10診断のアルコール依存症現在有病者: **57万人**
- 2013年推計: アルコール依存症生涯経験者 **107万人**（男性94万人、女性13万人）
- 2022年推計: アルコール依存症の疑いがある人 **64.4万人**

### 不眠症とアルコール依存の関連
- 不安を伴う不眠症がある場合、アルコール関連問題発症リスクは **2.32倍**
- 不眠→寝酒→耐性→飲酒量増加→依存の悪循環

## 睡眠時無呼吸症候群（SAS）への影響

### メカニズム
- アルコールは全身の筋肉を弛緩させる
- 特に咽頭・舌の筋肉弛緩 → 気道狭窄・閉塞
- いびきの重症度悪化
- 無呼吸低呼吸指数（AHI）の増加
- 最低酸素飽和度（LSAT）の低下

### 2020年メタ分析の知見
- アルコール摂取は:
  - いびきの重症度を悪化
  - 睡眠構造を変化
  - AHIを増加
  - LSATを低下
- 無呼吸の頻度だけでなく、1回あたりの無呼吸時間を大幅に延長

## 耐性と依存のメカニズム
- 寝酒の反復 → GABA受容体のダウンレギュレーション
- 同じ効果を得るために必要な飲酒量が増加（耐性）
- 寝酒をやめると反動で不眠が悪化（離脱症状）
- 離脱時のHPA軸活性化 → コルチゾール上昇 → さらなる不眠
- 典型的な悪循環: 不眠 → 寝酒 → 耐性 → 増量 → 依存 → 禁酒 → 離脱不眠 → 再飲酒

## アルコールの利尿作用
- 抗利尿ホルモン（ADH/バソプレシン）の分泌を抑制
- 夜間頻尿 → 睡眠中断
