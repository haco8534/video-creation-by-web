# YouTube 概要欄

## タイトル
記憶は「思い出す」たびに変わるという衝撃【脳科学で解明】

## サムネイル案
- 「記憶は嘘をつく」の文字 + 脳のイラスト + 割れたガラスのエフェクト

## 概要欄テキスト

あなたの一番大切な思い出、実は脳がこっそり書き換えているかもしれません。

2000年のNaderの発見により、長期記憶が「思い出すたびに不安定になる」ことが判明しました。この「記憶の再固定化」のメカニズム、Loftusの誤情報効果実験、DNA鑑定で判明した冤罪の真実、9/11のフラッシュバルブ記憶の幻想など、記憶にまつわる衝撃の科学をわかりやすく解説します。
