# 「部分痩せ」は物理的に不可能──5000回の腹筋が証明した残酷な事実

「腹筋でお腹の脂肪が落ちる」──この常識、科学では完全に否定されています。
5004回の腹筋実験、片腕12週間トレーニング、テニスプレーヤー研究……数十年の研究が繰り返し証明してきた「部分痩せ不可能」の科学的メカニズムと、本当に効果のあるダイエット法を徹底解説します。
