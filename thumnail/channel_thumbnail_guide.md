# サムネイルプロンプト生成リファレンス

> 台本を読み → サムネイル用プロンプトを複数パターン提案するためのリファレンス

---

## チャンネル概要

- **ジャンル**: 心理学・科学・社会の「通説を覆す」解説動画
- **方針**: ファクトフル。一次情報・裏付けのある事実ベース。難しいテーマを楽しく届ける
- **キャラ**: ずんだもん（解説）、めたん（聞き役）
- **視聴者**: 20〜40代、知的好奇心が強い層

---

## デザインの方向性

### 大原則

**「事実がドラマチックなら、ドラマチックに出す」**

インパクト＝印象操作ではない。台本が「ほぼ間違いだった」と言っているなら、それをそのまま力強く見せるのはファクトフル。ただし台本に書いていないことを盛ったり、ミスリードするキャッチは禁止。

### スタイル

```
✅ やること
- テキストが主役。画面の中央に大きく、高コントラストで
- イラスト寄り（フラット、線画、シルエット）
- 暗い背景（ダークネイビー、黒、ダークパープル）でコントラスト確保
- ビジュアルは添え物。テキストを引き立てる存在
- デフォルメ強めのキャラやシルエット
- 色はパキッと。赤、黄色、白をガツンと使う

❌ やらないこと
- 写実・フォトリアル
- AIっぽいキラキラ（パーティクル、光の粒子、レンズフレア、グロー）
- パステルカラーや柔らかいトーン（コントラスト不足）
- 台本に書いてない誇張（「全部ウソ」「衝撃の真実」等の盛り）
- おとなしいコピー（「誤解されてました」「〜の話」等の弱い表現）
```

### カラー

| 用途 | 色 |
|---|---|
| メインテキスト | **赤文字＋白太縁取り** / **白文字＋赤太縁取り** / **黄色文字＋黒太縁取り** |
| サブテキスト | 黄色 / 白 |
| 背景 | **真っ黒 / ダークネイビー / ダークパープル**（高コントラスト前提） |
| 対比背景 | **真っ赤 vs 真っ青** のベタ塗りフラット分割 |

### テキストのルール

- **メインキャッチ**: 8〜15文字。画面の主役。太ゴシック体、超特大
- **サブテキスト**: 5〜15文字。データや補足。小さめ or 帯・吹き出し内
- キャッチは**台本の冒頭フックやコアメッセージから**抽出する（勝手に盛らない）
- 動画タイトルとそのまま同じにしない（別角度の表現にする）
- **テキストのサイズ差**でインパクトを出す（小さい通説 → 巨大な否定）

## 台本から何を読み取るか

| 抽出する情報 | 台本のどこにあるか | 使い方 |
|---|---|---|
| **冒頭フック** | ヘッダー or Block 1冒頭 | → メインキャッチの最有力候補 |
| **コアメッセージ** | ヘッダー | → メインキャッチ or サブテキスト |
| **通説** | 前半で紹介される「常識」 | → 小さく書いて否定する / 対比の片側 |
| **真実** | 後半で明かされる結論 | → メインキャッチ |
| **衝撃データ** | 数字・統計 | → サブテキスト / データ型キャッチ |
| **象徴モチーフ** | テーマから連想 | → 添え物イラスト |

---

## キャッチコピーのパターン

台本の**フック文やコアメッセージ**をベースに、以下のパターンで変換する：

| パターン | テンプレート | 例 |
|---|---|---|
| 直球否定 | 「○○は無意味だった」 | 「我慢は無意味だった」 |
| ほぼ否定 | 「この実験、ほぼ間違いです」 | （台本フックそのまま） |
| 本当の答え | 「○○じゃなかった」 | 「"自制心"じゃなかった」 |
| データ提示 | 「○○したら○○が消えた」 | 「900人で追試したら効果が消えた」 |
| 疑問投げかけ | 「○○って本当？」 | 「我慢できる子が成功するって本当？」 |
| 対比 | 「○○じゃない　○○だった」 | 「才能じゃない 環境だった」 |

**NGコピー（台本に基づかない盛り）**:
- 「全部ウソ」「衝撃の真実」「ヤバすぎる」「まだ信じてるの？」

## イラスト要素の引き出し

### テーマ → モチーフ変換

```
【心理学実験】
マシュマロ実験 → 子ども＋テーブル＋白い丸、皿、ベル
ミルグラム実験 → 電気スイッチ、手が震える人
監獄実験 → 鉄格子、鍵、制服シルエット
認知バイアス → 歪んだ鏡、傾いた天秤
ダニングクルーガー → 小さい山の頂上で喜ぶ人

【概念】
通説の崩壊 → ×マーク、取り消し線、割れるもの
環境の影響 → 2つの部屋の対比（豪華vs殺風景）
成功 → 階段、トロフィー
データ否定 → グラフに×、フローチャートに×
```

### イラストスタイルの指示語

```
フラット系:    "flat illustration, minimal, simple shapes, bold colors"
線画系:        "line art, simple white outlines on dark background"
シルエット系:  "silhouette, flat color, high contrast"
教科書図解風:  "editorial illustration, infographic style, clean vectors"
```

## 完成例：マシュマロ実験

### 案A：吹き出しドカン型

```
背景：真っ黒に近いダークネイビー

中央左：線画イラストの子ども（デフォルメ強め）
        テーブルに頭を抱えて悩んでいるポーズ
        テーブルの上に白い丸が1つ
        白い線画、背景に対してくっきり

中央右（子どもの横からドカンと出る巨大吹き出し）：
太ゴシック体 特大
赤文字＋白太縁取り
「我慢は
　無意味だった」

下部：
黄色文字（中サイズ）
「世界一有名な心理学実験の誤解」

--ar 16:9
--style raw
--q 2
```

### 案B：左右対比型

```
背景：左半分＝真っ赤、右半分＝真っ青
      ベタ塗り、パキッと分割

左側：豪華な部屋のフラットイラスト（本棚、ランプ）
      椅子に座る子どものシルエット（白抜き）
右側：がらんとした部屋のフラットイラスト（壁と床だけ）
      同じポーズの子どものシルエット（白抜き）

中央（両側にまたがるように）：
太ゴシック体 特大
白文字＋黒太太縁取り（6px）
「差を生んだのは
　"自制心"じゃなかった」

--ar 16:9
--style raw
--q 2
```

### 案C：テキスト勝負型

```
背景：真っ黒

中央下にさりげなく：
線画イラストの皿とマシュマロ（小さめ、白い線のみ）
皿の上のマシュマロに×印

上段（小さめ、白文字）：
「"我慢できる子が成功する"」

下段（超巨大、赤文字＋白太縁取り）：
「ほぼ間違いです」

最下部（中サイズ、黄色文字）：
「──マシュマロ実験 900人の追試結果──」

--ar 16:9
--style raw
--q 2
```

```

中央：教科書風フラットイラスト

      のフローチャート
      その上からデカい赤の×マークがバサッ

×の下（大きく）：
太ゴシック体 特大
黄色文字＋黒太縁取り
「900人で追試したら
　　効果が消えた」

--ar 16:9

```
